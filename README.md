# MILP Cluster Analysis

**Десктопное приложение для кластерной линейной регрессии на базе смешанного целочисленного линейного программирования (MILP)**

</div>

---

## О проекте

Приложение разбивает двумерные данные на *K* кластеров и строит **отдельную линейную регрессию** для каждого кластера так, чтобы суммарная средняя ошибка была минимальна. Разбиение и коэффициенты регрессии находятся **одновременно** — как решение единой MILP-задачи с помощью открытого решателя [HiGHS](https://highs.dev).

Основные возможности:

- **Автоматическая кластеризация** с настраиваемым числом кластеров
- **Два режима разбиения**: без ограничений и с монотонным порядком по x
- **Интерактивные Plotly-графики** кластеров и ошибок
- **Экспорт отчёта в PDF** с уравнениями регрессии и метриками
- **Glassmorphism UI** — тёмная и светлая темы

- Установленные системные зависимости для PySide6 (Qt 6)

### Шаги


## Математическая постановка

Пусть дано $n$ точек $(x_i, y_i)$ и число кластеров $K$.

Нужно найти:
- **бинарные переменные** $z_{ik} \in \{0,1\}$ — принадлежность точки $i$ кластеру $k$
- **коэффициенты** $a_k, b_k$ линейной регрессии $\hat{y} = a_k x + b_k$ для каждого кластера

Минимизируя:

$$E = \frac{1}{n} \sum_{i=1}^{n} \sum_{k=1}^{K} z_{ik} \cdot |y_i - a_k x_i - b_k|$$

При ограничениях:
- $\sum_{k=1}^{K} z_{ik} = 1$ — каждая точка принадлежит ровно одному кластеру
- (опц.) монотонный порядок кластеров по $x$

Абсолютное значение линеаризуется стандартным приёмом через вспомогательные переменные и Big-M.
